Question-1: What are the different ways to select an element in the DOM?

Answer-1: You can select DOM elements using methods like `document.getElementById` for a single element, `document.getElementsByClassName` and `document.getElementsByTagName` for collections, and `document.querySelector/querySelectorAll` for dynamic queries by CSS selectors. More sophisticated methods like `document.evaluate` for XPath expressions and DOM traversal methods like `closest` and `childNodes` provide more control over element selection.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer-2: `innerHTML` returns the HTML content of an element as a string, allowing you to insert HTML elements and modify the structure. `innerText` returns the visible text content of an element, including whitespace and line breaks, but excludes hidden elements. `textContent` returns the complete text content of an element, including hidden elements and whitespace, making it more reliable for cross-browser compatibility.

Question-3: What is event delegation in the DOM?

Answer-3: Event delegation is a technique where you attach an event listener to a parent element instead of each child element. This approach is efficient for handling events on dynamically added elements and reduces the need for attaching listeners to each element.

Question-4: What is event bubbling in the DOM?

Answer-4: Event bubbling is a process where an event is triggered on a child element and then propagated up to its parent elements, allowing you to handle events at the parent level. This is a common pattern in event handling and can be used to implement event delegation and handle events on dynamically added elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer-5: You can create elements using `document.createElement`, add elements using `appendChild` or `insertAdjacentElement`, and remove elements using `removeChild` or `remove`.
